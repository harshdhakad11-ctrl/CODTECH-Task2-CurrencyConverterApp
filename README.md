# Project Description

This Android application allows users to convert one currency into another using live exchange rates fetched from an online API.

The application is developed using Java in Android Studio and stores conversion history locally using SQLite Database.

---

# Features

- Live Exchange Rates API
- Currency Selection
- Search Currency
- Currency Swap
- Conversion History
- SQLite Database
- Material UI
- Splash Screen
- Internet Connectivity Check

---

# Technologies Used

- Java
- XML
- Android Studio
- Volley Library
- SQLite Database
- ExchangeRate API

---

# Project Scope

This application helps users convert currencies quickly using real-time exchange rates.

The project demonstrates Android UI Design, REST API Integration, Local Database Management and Material Design.
